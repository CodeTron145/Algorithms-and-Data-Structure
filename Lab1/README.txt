Instrukcja do zad1.c

Opisanie: 
Ten program implementuje kolejkę FIFO. Dodaje nowy element na koncu, usuwa element na początku.

Wywołanie:

gcc zad1.c -o zad1
./zad1


Instrukcja do zad2.c

Opisanie: 
Ten program implementuje jednokierunkową listę. Wszystkie użycia funkcij są w mainie. Dlatego wystarczy poprostu wpisać do terminalu ./zad2 by zobaczyć działanie funkcij na liscie.

Wywołanie:

gcc zad2.c -o zad2
./zad2


Instrukcja do zad3.c
To samo, tylko dla dwukierunkowej listy cyklicznej.

Wywołanie:

gcc zad3.c -o zad3
./zad3
